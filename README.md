An Interactive Gallery that allows a user to serach up different Artists on Spotify and showcases all of their albums. 
Built with the Spotify API, React, and Framer Motion
Every album contains data such as song count, release date, and a link to the album, and mouse tracked motion effects

## üíª Technologies

- 'React'
- 'Javascript'
- 'Framer Motion'
- 'Vite'
- **Spotify Developer API**

## üìç Features

- Integrates the Spotify Web API to fetch artist and album data in real time
- Allows users to search for an artist and view their complete album discography
- Displays key album information, including release date and track count
- Provides direct links to each album on Spotify
- Features viewport-aware rendering so albums react when entering the screen
- Includes mouse-reactive interactions for dynamic visual feedback
- Designed with a responsive layout for use across different screen sizes
